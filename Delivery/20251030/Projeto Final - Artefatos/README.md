## Descrição do Tema Escolhido (o que foi feito)

SkyNET-I2A2: Processamento Fiscal Inteligente

Breve descrição:

Este trabalho apresenta o SkyNET-I2A2, uma solução para processamento fiscal inteligente que automatiza a extração, análise e verificação de documentos fiscais. Os artefatos disponíveis nesta pasta incluem a apresentação em PPTX e o vídeo demonstrativo do projeto.
